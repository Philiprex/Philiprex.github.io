Despite the existence of another package for the discrete Weibull, I wrote this one primarily to allow working with the location–scale parameterization (the Stute / Wikipedia parameterization) for a paper to be published in *Statistical Modelling*. It also supports the original parameterization first introduced for the discrete Weibull on Wikipedia.

The package is on GitHub: [Philiprex/dWeibullAlt](https://github.com/Philiprex/dWeibullAlt).
