This repository contains the data and code used in a statistical paper I co-authored, soon to be published in *Statistical Modelling*. It includes code for an empirical application of our framework to several states in the 2016 and 2020 U.S. presidential elections, including the Weibull-modeling-based distributional assumption verification and the discrete, one-sample, two-tail, bootstrap KS test.

The code lives at [Philiprex/L2-BL-10](https://github.com/Philiprex/L2-BL-10), and depends on my [dWeibullAlt]({% post_url 2026-01-20-dweibullalt-stats-package %}) package.
